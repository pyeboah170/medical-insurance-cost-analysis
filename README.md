# medical-insurance-cost-analysis

This project presents an exploratory analysis of how age, BMI, and sex are associated with estimated medical insurance costs using a simplified linear model.


## Key Concepts
- Exploratory data analysis (EDA)
- Correlation vs. causation
- Ethical considerations in healthcare data
- Model-based interpretation

## Tools Used
- Python
- Jupyter Notebook

## Notes
This analysis reflects assumptions embedded in the dataset and model and does not establish real-world causal relationships.
